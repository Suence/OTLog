## ✨ 功能

- 🐱‍💻 使用 .NET 5 WPF 开发
- ☁️ 自动更新
- 🚫 无任何联网功能
- ✔️ 智能提醒
- 🎨 浅色/深色主题自由切换(可跟随 Windows10 系统主题)
- 🙋 针对每个人提供个性化通知
- 🛠 更多功能开发中
